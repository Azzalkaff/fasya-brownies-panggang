# Concept Web

## Brownies Panggang Premium

Version: 0.1

---

# 1. Project Overview

Website landing page premium untuk brand brownies panggang.

Tujuan utama website adalah:

* Membangun kredibilitas brand.
* Menampilkan produk secara profesional.
* Mengarahkan pengunjung untuk melakukan pemesanan.
* Menjadi pusat informasi dan traffic hub untuk seluruh channel penjualan.

Website tidak difokuskan sebagai e-commerce pada fase awal.

Website berfungsi sebagai:

* Brand Website
* Product Showcase
* Local SEO Asset
* Conversion Landing Page

---

# 2. Business Goals

## Primary Goal

Meningkatkan jumlah order melalui WhatsApp.

## Secondary Goals

* Meningkatkan kepercayaan calon pelanggan.
* Memperkuat branding premium.
* Mendukung SEO lokal.
* Mengarahkan traffic ke marketplace.

## Success Metrics

* Klik tombol WhatsApp.
* Scroll depth.
* Waktu kunjungan.
* Klik marketplace.
* Klik Instagram.
* Peningkatan pencarian brand.

# 3. Target Audience

## Primary Audience

Pelanggan lokal kota operasional.

Karakteristik:

* Mencari brownies untuk konsumsi pribadi.
* Mencari oleh-oleh.
* Mencari hampers.
* Membeli untuk keluarga.

## Secondary Audience

Pembeli dari luar kota.

Karakteristik:

* Mencari brownies premium.
* Memesan melalui marketplace.
* Membutuhkan pengiriman.

### Premium Ingredients

Menggunakan bahan pilihan dengan rasa cokelat yang kaya.

### Perfect For Gifts

Cocok untuk hadiah, hampers, dan momen spesial.

### Trusted By Customers

Telah dinikmati oleh banyak pelanggan.

## Layout

# 20. Definition of Success

Website berhasil apabila:

* Pengunjung memahami produk dalam kurang dari 10 detik.
* Pengunjung dapat menemukan tombol order tanpa kebingungan.
* Website terlihat premium dan terpercaya.
* Loading cepat pada perangkat mobile.
* Meningkatkan jumlah order yang masuk melalui WhatsApp.
